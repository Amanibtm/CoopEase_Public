# CoopEase: Intelligent Educational Resource Scheduler

A Django web application that automates timetable and resource allocation for schools and educational establishments using Google OR-Tools constraint solver.

**Key Features**
- Handles complex constraints (teacher availability, room capacity, time conflicts)
- Reduces manual scheduling time by ~90% (from 15+ days to ~30 minutes)

**Technologies**
Python • Django • Google OR-Tools • PostgreSQL

**Status**  
Master's final project (developed solo). Originally deployed on Render.
